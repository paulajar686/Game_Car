# Game_Car
---

# **Car Game – Descripción del juego**

**Car Game** es un dinámico y adictivo juego de conducción en 2D, desarrollado con Pygame, donde el jugador debe esquivar vehículos mientras avanza por una carretera infinita. El objetivo es sobrevivir el mayor tiempo posible, evitando colisiones y acumulando la mayor cantidad de puntos.

---

# **Mecánica del juego**
- El jugador controla un coche que puede cambiar de carril utilizando las teclas **← (izquierda)** y **→ (derecha)**.
- Otros vehículos aparecen en carriles aleatorios y se mueven hacia abajo.
- Si el coche del jugador colisiona con otro vehículo, se produce un **choque y el juego termina**.
- Con cada vehículo esquivado, el jugador gana puntos.
- A medida que se acumulan puntos, la **velocidad del juego aumenta** y se sube de nivel.

---

# **Funciones adicionales**
- Sistema de **niveles dinámico**: el nivel sube cada 10 puntos, aumentando la dificultad.
- **Power-ups**:
  - **Turbo**: aumenta la velocidad temporalmente.
- Sistema de **usuarios**:
  - Puedes seleccionar un usuario registrado o crear uno nuevo.
  - El nombre y la puntuación se guardan automáticamente.
- **Ranking de puntuaciones**: al finalizar una partida, se muestra el top 5 de mejores puntajes registrados.
- **Menú de inicio**: acceso simple e intuitivo para comenzar o salir del juego.

---

# **Requisitos**
- Python 3
- Pygame
- Carpeta `images/` con los sprites del coche, vehículos y power-ups.
